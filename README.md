# PromoScan

PromoScan is a web application designed to help users find the best deals, coupons, and cashback offers for any product instantly. By simply scanning a product's barcode, users can unlock a variety of savings and rewards, making their shopping experience smarter and more economical.

## Features

- **Product Scanning:** Instantly scan product barcodes to fetch information.
- **Benefit Aggregation:** Finds and displays relevant coupons, cashback offers, and loyalty rewards.
- **Digital Wallet:** Keeps track of your saved offers and rewards.
- **User Authentication:** Secure login and user management.

## Pages

The application consists of the following pages:

- `login.html`: Welcome and login page for user authentication.
- `setup.html`: User setup and onboarding page.
- `scan.html`: The main page for scanning product barcodes.
- `scan-complete.html`: Displays the product after a successful scan.
- `benefits-results.html`: Shows the aggregated benefits and offers for the scanned product.

## Technologies Used

- **Frontend:** HTML, Tailwind CSS, JavaScript
- **Icons:** Font Awesome

## How to Run

1.  Clone this repository to your local machine.
2.  Open any of the `.html` files in your web browser to view the application.

There is no build process required as the project is built with simple HTML, CSS, and JavaScript.
